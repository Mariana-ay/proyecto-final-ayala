Tech Store
Pequeña página de ejemplo para una tienda y servicio de reparación de celulares. Está construida con Bootstrap y SASS (SCSS). El proyecto incluye una estructura modular de SCSS (partials) y un CSS compilado listo para usar en desarrollo.

Características
Layout responsive con Bootstrap.
SCSS modular separado en partials (_variables, _mixins, _base, _footer, _body).
Componentes: navbar, cards de productos, tarjetas de servicios, botón flotante de WhatsApp y footer.
Animaciones básicas: transición hover en cards, animación "float" para el botón WhatsApp.
Usos de variables, mixins.
